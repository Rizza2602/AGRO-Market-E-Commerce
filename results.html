Key Issues

Identified Problems

• Some users feel they might need technical support (Q4 avg: 2.65 — moderate concern)

• A portion of respondents found learning curve higher than ideal (Q10 avg: 3.20)

• Inconsistency in UI noted by some users (Q6 avg: 2.25)

• Navigation clarity needed for first-time users

• Mobile responsiveness and loading speed concerns raised in open-ended feedback

Recommendations

Improvement Suggestions

  • ✅ Add a brief onboarding tutorial or tooltips for first-time users
  • ✅ Include language selection (English, Tagalog, Bicolano) via dropdown
  • ✅ Add map/tracking feature for product delivery
  • ✅ Provide default product image options to ease listing
  • ✅ Optimize mobile image loading speed
  • ✅ Add location-based filtering for buyers to find nearby sellers
  • ✅ Expand product categories for better browsing

Discussion

Findings Analysis

The Agro Market E-Commerce prototype achieved an average SUS score of 67.0 out of 100, placing it in the "marginally acceptable" range — just below the industry average of 68. While users consistently praised the system's ease of use, simple layout, and convenience (Q3: 3.80, Q5: 4.10, Q9: 4.00), concerns remain around the learning curve (Q10: 3.20) and whether some users might need technical assistance (Q4: 2.65). Negatively-worded items scored low (meaning users generally did not find it overly complex or cumbersome), which is a positive signal. To push the SUS score above 80 and reach a "Good" grade, improvements in onboarding, mobile performance, language support, and delivery tracking are recommended based on direct user feedback.
